How to upload your model for evaluation

Single-Model

Single Model Form Sample Download

To upload your model you must insert a .json or .txt file with a json format. Each object contains 2 variables, name and elements. The name variable is the name you give your model, the elements variable contains objects which correspond to a single confusion matrix. Each 'elements' object contains 5 variables, True Positive; tp, False Positive; fp, False Negative; fn, True Negative; tn and epoch. You can follow the format on the image below or download our Single Model Form Sample.
