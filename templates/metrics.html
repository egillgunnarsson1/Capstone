What metrics are we using?

Accuracy

\[acc = {{tp + tn} \over {p + n}}\]

Balanced Accuracy

\[bac = {1 \over 2}({{tp + tn} \over {p + n}})\]

Geometric Mean

\[gem = ({{tp \over p} * {tn \over n}})^{1 \over 2}\]

Recall

\[rec = {{tp} \over {p}}\]

F1-Score

\[f{_1} = {2*{{pre * rec} \over {pre + rec}}}\]

Gilbert's Skill Score

\[gss = {{tp - r} \over {tp + fp + fn - r}}\]

Doolittle's Skill Score

\[dss = {({tp * tn - fp * fn})^2 \over {(tp + fp) * (tp + fn) * p * n}}\]

True Skill Statistic

\[tss = {{tp \over p} - {fp \over n}}\]

Heidke Skill Score

\[hss = {{2*((tp * tn) - (fn * fp))} \over {p * (fn + tn) + n * (tp + fp)}}\]

Youden Index

\[ydn = {{tp * tn - fn * fp} \over {(tp + fn) * (fp + tn)}}\]

ROC Space

\[ROC{_d} = ({({fp \over n})^2 + ({fn \over p})^2})^{1 \over 2}\]
